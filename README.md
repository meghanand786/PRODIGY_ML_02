# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 Project Overview
Create a K-means clustering algorithm to group customers of a retail store based on their purchase history.
---

## 📂 Dataset
- Source: Kaggle  
- Dataset Name: Mall Customer Segmentation Data  
- Features used:
  - Annual Income (k$)
  - Spending Score (1-100)

---

## 🎯 Objective
To identify different customer segments such as:
- High Income – High Spending
- High Income – Low Spending
- Low Income – High Spending
- Low Income – Low Spending
- Average Customers

---

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## ⚙️ Implementation Steps

1. Data Loading and Exploration  
2. Feature Selection  
3. Data Preprocessing (Scaling)  
4. Finding Optimal Clusters using Elbow Method  
5. Applying K-Means Clustering  
6. Visualizing Customer Segments  

---

## 📊 Model Details

- Algorithm: K-Means Clustering  
- Optimal Clusters: 5 (determined using Elbow Method)  
- Distance Metric: Euclidean Distance  

---

## 📈 Results
The model successfully segmented customers into distinct groups, helping businesses:
- Target the right audience  
- Improve marketing strategies  
- Increase customer retention
